Información útil sobre el juego:
  • Cómo jugar.
  • Créditos.

Instalación: realmente, no requiere instalación, simplemente descarga el archivo, descomprime y empieza a jugar.
  • Windows: simplemente descomprime el archivo .zip de la descarga. Una vez descomprimida, haz doble click sobre el fichero Farmacos_juego.exe para empezar a jugar
    • Es posible que Windows Defender detecte el archivo ejecutable como una amenaza. No te preocupes, esto lo hace de forma habitual con programas de origen desconocido.
    • Si es así, recibirás una notificación de Windos Defender. Clicka sobre ella y podrás seleccionar añadir como una excepción el programa.
  • Linux: descomprime el archivo .zip de la descarga. En la carpeta "data" hay dos archivos ejecutables: ags32 (para sistemas de 32 bits) y ags64 (para sistemas de 64 bits).
    • Tendrás que dar permiso de ejecución. Para ello:
    • Abre un terminal y navega a la carpeta data
    • Cambia los permisos de ejecución del fichero ags32 o ags64. Puedes usar la siguiente línea de comando: chmod ugo+x ags64 ags32.

Si quieres mejorar o cambiar el juego, hemos creado un repositorio para desarrolladores. Puedes descargarlo aquí. Para poder realizar modificaciones, necesitarás descargar Adventure Game Studio.
